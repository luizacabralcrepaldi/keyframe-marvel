O meu projeto foi desenvolvido com o tema Marvel e teve como principal objetivo praticar animações utilizando HTML e CSS, principalmente com o uso de `@keyframes`. A ideia do site foi criar uma página moderna e interativa inspirada no universo Marvel, onde fãs possam acompanhar novidades, filmes e notícias relacionadas aos personagens e ao multiverso Marvel.

Durante o desenvolvimento, utilizei diversas animações para deixar o site mais dinâmico, como fundo animado, texto neon, imagem flutuando, loader personalizado e botões pulsando. Também criei um carrossel automático usando apenas CSS e Keyframes, sem utilizar JavaScript.

Além disso, apliquei efeitos de hover, zoom e sombras neon para melhorar a experiência visual do usuário. Com esse projeto, consegui aprender melhor como funcionam propriedades como `transform`, `translate`, `scale`, `rotate` e `animation`, entendendo como as animações são usadas em sites reais para deixar as páginas mais modernas e profissionais.
